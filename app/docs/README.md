# app/docs/ — 프로젝트 공개 문서

GitHub Pages로 배포되는 공개 문서 폴더입니다.

## 구조

```
docs/
├── images/             분석 차트, 아키텍처 다이어그램 등
├── README.md
```

## 포함 대상

- 프로젝트 아키텍처 다이어그램
- 분석 결과 시각화 이미지 (About/Research 페이지용)
- API 문서
- 사용자 가이드

## 주의

- PRD, 내부 스펙 등 비공개 문서는 루트 `docs/` (gitignore) 에 보관
- 이미지는 **1MB 이하** 권장, WebP 포맷 선호
