             • 펄싱 아우라
             • 국가 라벨
             • 효과도 바
             → 크기 7.5배 증가
             → 시각적으로 분명
             → 4가지 애니메이션
             
  PM2.5 마커: 구체 0.01 + 링 ✅
             • 반투명
             • 부드러운 펄싱
             → 배경으로 표현
